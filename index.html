HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • span is used to seperate text only within your html file so that it can be targeted for css.
  • b between brackets is used inline to bold certain letters or words with an element without using css.
  • img is used to display an image with the html file. Requires a src attribute.
  • a is used to anchor a link within the html file. Requires an href attribute with the url.
  • The Class attribute is used so that the element that the class is attached to can be targeted for styling using CSS.

CSS

  • Cascading Style Sheets, or CSS is used to style a webpage.
  • Margin indicates how much space is around a given container.
  • Padding indicates how much space is inside the container.
  • text-decoration is used to decorate the text such as, underlining, crossing the text out, etc.
  • background-color gives the background of an element a color using rgb, hexadecimal, or hsl colors.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it
  • When Git asks for username and password: git remote set-url origin (ssh key)
  • To commit files for pull: git commit -m “Message about what I edited”
  • To push committed files to the repo on GitHub: git push origin (name of branch)
  • To add files to commit stage: git add -A

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Variables can hold many types of information such as, strings, integers, or any kind of data structure.
  • Functions are a set of instructions that must be called on using their given function name.
  • Control flow is the order in which a computer executes code in a script.
