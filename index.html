Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it
  • When Git asks for username and password: git remote set-url origin (ssh key)
  • To commit files for pull: git commit -m “Message about what I edited”
  • To push committed files to the repo on GitHub: git push origin (name of branch)

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Variables can hold many types of information such as, strings, integers, or any kind of data structure.
  • Functions are a set of instructions that must be called on using their given function name.
  • Control flow is the order in which a computer executes code in a script.
